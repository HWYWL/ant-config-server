# ant-config-server
> 蚂蚁配置中心

		用于管理项目中常改动的配置而无需改动代码或者重启服务器，我们的线上Properties文件。我们在前端配置我们需要配置的数据然后存储于数据库中，
	这里使用的是SQLite数据库，在系统初始化时会自动生成无需手动添加。当我们在其他系统需要用到配置文件时，会通过Http请求获取相应的配置，无需向更改Properties文件一样重启服务器，
	一旦更改即时生效。
